# 🧮 Math Master

A high-speed, interactive math quiz challenge built with React and Vite. Test your arithmetic skills across multiple difficulty levels, track your streaks, and aim for a perfect score!

## ✨ Features

- **🎮 Dynamic Gameplay**: Choose between Timed and Unlimited modes.
- **📈 Progressive Difficulty**: Easy, Medium, and Hard levels to challenge your brain.
- **🔥 Streak System**: Keep the momentum going with bonus points for consecutive correct answers.
- **🎨 Modern UI**: Vibrant design with glassmorphism effects, smooth animations, and interactive feedback.
- **📊 Detailed Results**: Comprehensive performance breakdown after every session.
- **⚡ Fast & Responsive**: Built on Vite for lightning-fast HMR and performance.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (version 18 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```bash
   cd Booking-system
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Running the Application

Start the development server:
```bash
npm run dev
```
The app will be available at `http://localhost:5173`.

## 🛠️ Built With

- **React 19** - UI Framework
- **Vite** - Build Tool & Dev Server
- **CSS3** - Styling with modern animations
- **JavaScript (ES6+)** - Logic and Question Generation

## 📂 Project Structure

- `src/components`: React components (`MathQuiz`, `MathQuestionCard`, `ScoreBoard`, etc.)
- `src/utils`: Game logic and mathematical question generators.
- `src/styles`: Visual styles and theme definitions.
